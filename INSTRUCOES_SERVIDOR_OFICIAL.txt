================================================================================
                    SOLUÇÃO DE PROBLEMAS
================================================================================

1. "Python não encontrado"
   - Instale o Python 3.11+ e adicione ao PATH

2. "Não foi possível conectar ao banco"
   - Certifique-se de que o PostgreSQL está rodando
   - Execute o script como Administrador
   - Verifique se o banco "monpec_oficial" existe

3. "manage.py não encontrado"
   - Execute o script na raiz do projeto Monpec

4. "Porta 8000 ocupada"
   - O script automaticamente libera a porta
