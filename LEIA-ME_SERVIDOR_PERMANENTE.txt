========================================
MONPEC - SERVIDOR PERMANENTE
Sistema de Gestão Rural - Desenvolvimento
========================================

ARQUIVOS CRIADOS:
=================

1. MONPEC DESENVOLVIMENTO.bat
   - Inicia o servidor Django em modo desenvolvimento
   - Usa: sistema_rural.settings (DESENVOLVIMENTO)
   - Servidor disponível em: http://127.0.0.1:8000/
   - Para parar: Feche a janela ou pressione Ctrl+C

2. MONPEC_DESENVOLVIMENTO_PERMANENTE.bat
   - Servidor com auto-restart (reinicia automaticamente se parar)
   - Mantém o servidor sempre online
   - Ideal para uso contínuo
   - Para parar: Pressione Ctrl+C duas vezes

3. INSTALAR_SERVIDOR_PERMANENTE.bat
   - Instala o servidor como tarefa agendada do Windows
   - Inicia automaticamente quando você faz login
   - PRECISA ser executado como Administrador
   - Clique com botão direito > "Executar como administrador"

4. REMOVER_SERVIDOR_PERMANENTE.bat
   - Remove a tarefa agendada do Windows
   - Para todos os processos Python do MONPEC
   - PRECISA ser executado como Administrador

COMO USAR:
==========

OPÇÃO 1 - Iniciar Manualmente (Simples):
-----------------------------------------
1. Clique duas vezes em: "MONPEC DESENVOLVIMENTO.bat"
2. O servidor iniciará e ficará rodando
3. Para parar: Feche a janela

OPÇÃO 2 - Servidor Permanente com Auto-Restart:
------------------------------------------------
1. Clique duas vezes em: "MONPEC_DESENVOLVIMENTO_PERMANENTE.bat"
2. O servidor iniciará e reiniciará automaticamente se parar
3. Para parar: Pressione Ctrl+C duas vezes

OPÇÃO 3 - Instalar como Serviço do Windows (Recomendado):
----------------------------------------------------------
1. Clique com botão direito em: "INSTALAR_SERVIDOR_PERMANENTE.bat"
2. Selecione: "Executar como administrador"
3. O servidor será instalado e iniciará automaticamente no login
4. Para remover: Execute "REMOVER_SERVIDOR_PERMANENTE.bat" como administrador

CONFIGURAÇÃO:
=============
- Settings: sistema_rural.settings (DESENVOLVIMENTO)
- Porta: 8000
- URL: http://127.0.0.1:8000/ ou http://localhost:8000/
- Página inicial: Landing page com vídeo
- Página de login: Login com card promocional

VERIFICAR SE ESTÁ RODANDO:
===========================
1. Abra o navegador e acesse: http://127.0.0.1:8000/
2. Você deve ver a landing page com vídeo
3. Ou acesse: http://127.0.0.1:8000/login/
4. Você deve ver a página de login com card promocional

PARAR O SERVIDOR:
=================
- Se estiver rodando manualmente: Feche a janela do CMD
- Se estiver como serviço: Execute REMOVER_SERVIDOR_PERMANENTE.bat
- Ou use o Gerenciador de Tarefas para finalizar processos python.exe

OBSERVAÇÕES:
============
- O servidor usa Python local (python311\python.exe) se disponível
- Caso contrário, usa o Python do sistema
- Certifique-se de que o Python está instalado e funcionando
- O banco de dados SQLite será usado automaticamente

SUPORTE:
========
Em caso de problemas:
1. Verifique se o Python está instalado
2. Verifique se o arquivo manage.py existe na pasta
3. Verifique se a porta 8000 está livre
4. Execute o script de verificação: verificar_processos_sistema.ps1
