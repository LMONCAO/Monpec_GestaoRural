================================================================================
INSTRUCOES: APOS GERAR NOVA PROJECAO
================================================================================

IMPORTANTE: Ao gerar uma nova projecao, algumas correcoes podem ser perdidas.
Siga estes passos para garantir que tudo fique correto:

1. GERE A NOVA PROJECAO normalmente pelo sistema

2. EXECUTE O SCRIPT DE VERIFICACAO:
   - Execute: EXECUTAR_APOS_NOVA_PROJECAO.bat
   - Ou execute manualmente os scripts na ordem:
     a) verificar_vinculacao_correcoes_planejamento.py
     b) garantir_correcoes_apos_nova_projecao.py
     c) verificar_saldos_negativos_todas_fazendas.py
     d) verificar_balanceamento_transferencias.py

3. VERIFIQUE OS RESULTADOS:
   - Nenhum saldo negativo deve aparecer
   - Todas as transferencias devem estar balanceadas
   - Se houver problemas, os scripts vao tentar corrigir automaticamente

4. SE NECESSARIO, EXECUTE SCRIPTS ESPECIFICOS:
   - Se faltarem transferencias: garantir_correcoes_apos_nova_projecao.py
   - Se faltarem evolucoes no Girassol: corrigir_completo_girassol_todas_transferencias.py
   - Se faltarem vendas no Girassol: criar_vendas_90_dias_apos_evolucao_girassol.py

================================================================================
CORRECOES QUE DEVEM SER MANTIDAS:
================================================================================

1. TRANSFERENCIAS:
   - Canta Galo -> Invernada Grande: 512 vacas descarte (2022)
   - Canta Galo -> Favo de Mel: Todas as transferencias de garrotes
   - Favo de Mel -> Girassol: Todas as transferencias de garrotes

2. EVOLUCOES E VENDAS NO GIRASSOL:
   - Todas as evolucoes de garrotes para bois (12 meses apos entrada)
   - Todas as vendas de bois (90 dias apos evolucao)

3. SALDOS:
   - Nenhum saldo negativo em nenhuma fazenda
   - Todas as transferencias devem ter entrada correspondente

================================================================================
SCRIPTS DISPONIVEIS:
================================================================================

- verificar_vinculacao_correcoes_planejamento.py
  Verifica se as correcoes estao vinculadas ao planejamento atual

- garantir_correcoes_apos_nova_projecao.py
  Recria correcoes necessarias apos nova projecao

- verificar_saldos_negativos_todas_fazendas.py
  Verifica se ha saldos negativos em todas as fazendas

- verificar_balanceamento_transferencias.py
  Verifica se todas as transferencias estao balanceadas

- corrigir_completo_girassol_todas_transferencias.py
  Cria evolucoes e vendas para todos os garrotes no Girassol

- criar_vendas_90_dias_apos_evolucao_girassol.py
  Cria vendas 90 dias apos as evolucoes no Girassol
