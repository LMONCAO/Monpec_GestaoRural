# Como Cadastrar Múltiplas Fazendas

Este guia explica como cadastrar mais de uma fazenda (propriedade) no sistema MONPEC após criar um novo usuário.

## Opções para Acessar o Cadastro de Fazendas

### Opção 1: Menu Lateral (Mais Fácil)

1. **Acesse o Menu Lateral** - No lado esquerdo da tela, você verá vários módulos
2. **Procure por "Minhas Fazendas"** - No final do menu, você encontrará um item chamado "Minhas Fazendas"
3. **Clique em "Minhas Fazendas"** - Isso abrirá a página com todas as suas fazendas cadastradas
4. **Clique no botão "Adicionar Nova Propriedade"** - No topo da página, você verá este botão

### Opção 2: URL Direta

Você pode acessar diretamente pela URL:
```
http://seu-servidor/propriedades/
```

Isso abrirá automaticamente a lista de suas fazendas, onde você pode adicionar novas.

### Opção 3: A partir de uma Fazenda Existente

1. Se você já tiver uma fazenda cadastrada, acesse qualquer módulo dessa fazenda
2. No **cabeçalho superior**, você verá um seletor com o nome da fazenda atual
3. Clique no menu lateral em **"Minhas Fazendas"**
4. Na página que abrir, clique em **"Adicionar Nova Propriedade"**

## Processo de Cadastro

Ao clicar em "Adicionar Nova Propriedade", você será direcionado para um formulário onde deverá preencher:

### Campos Obrigatórios:

- **Nome da Propriedade** - Nome da fazenda (ex: "Fazenda São João")
- **Município** - Cidade onde a fazenda está localizada
- **UF** - Estado (2 letras, ex: SP, MG, MS)
- **Área Total (ha)** - Área total em hectares

### Campos Opcionais mas Recomendados:

- **Endereço** - Endereço completo da fazenda
- **CEP** - Código postal
- **Bairro/Distrito** - Bairro ou distrito
- **Tipo de Operação** - Geralmente "Pecuária"
- **Ciclos Pecuários** - Selecione os tipos de ciclo (Cria, Recria, Engorda, Ciclo Completo)
- **Tipo de Propriedade** - Própria ou Arrendamento
- **Latitude e Longitude** - Coordenadas geográficas (opcional)
- **Documentos** - NIRF, INCRA, CAR, Inscrição Estadual

### Documentos Adicionais (Opcional):

- **NIRF** - Número de identificação rural federal
- **INCRA** - Número do INCRA
- **CAR** - Cadastro Ambiental Rural
- **Inscrição Estadual** - Inscrição estadual da propriedade

## Após Cadastrar

Após preencher o formulário e clicar em "Salvar":

1. Você será redirecionado para a **lista de propriedades**
2. A nova fazenda aparecerá na lista
3. Você pode **clicar em "Acessar Funcionalidades"** para começar a usar a nova fazenda
4. Ou pode **alternar entre fazendas** usando o seletor no cabeçalho superior

## Dicas Importantes

1. **Múltiplas Fazendas**: Você pode cadastrar quantas fazendas precisar - não há limite
2. **Alternar entre Fazendas**: Use o seletor no cabeçalho superior para alternar rapidamente entre fazendas
3. **Editar Fazendas**: Na lista de fazendas, você pode clicar em "Editar" para modificar os dados de qualquer fazenda
4. **Excluir Fazendas**: Use com cuidado! A exclusão de uma fazenda removerá todos os dados associados

## Solução de Problemas

### "Você ainda não possui um produtor cadastrado"

Se você ver esta mensagem, significa que seu usuário ainda não foi vinculado a um produtor rural. Neste caso:
- Entre em contato com o administrador do sistema
- Ou verifique se você completou o processo de cadastro corretamente

### Não consigo ver "Minhas Fazendas" no menu

Se você não vê o item no menu:
1. Certifique-se de que está logado no sistema
2. Verifique se você tem uma propriedade cadastrada (mesmo que seja só uma)
3. Tente acessar pela URL direta: `/propriedades/`

## Suporte

Em caso de dúvidas ou problemas:
1. Verifique se você está logado com o usuário correto
2. Certifique-se de ter permissões para cadastrar propriedades
3. Entre em contato com o suporte técnico se o problema persistir
