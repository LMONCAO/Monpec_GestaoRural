================================================================================
COMO VISUALIZAR OS DADOS CARREGADOS NO DASHBOARD FINANCEIRO
================================================================================

Os dados foram carregados para o ano de 2022, mas o dashboard está filtrando
para novembro de 2025. Para visualizar os dados carregados, você precisa
ajustar o filtro de data no dashboard.

================================================================================
OPÇÃO 1: AJUSTAR O FILTRO DE DATA NO DASHBOARD
================================================================================

1. No dashboard financeiro, localize o filtro de data no topo da página
   (campos "Data Início" e "Data Fim")

2. Altere as datas para:
   - Data Início: 01/01/2022
   - Data Fim: 31/12/2022

3. Clique no botão "Filtrar"

4. Agora você verá:
   - Receitas e Despesas do período 2022
   - Gráficos de fluxo de caixa de 2022
   - Despesas por fornecedor e centro de custo
   - Todos os lançamentos de 2022

================================================================================
OPÇÃO 2: CARREGAR DADOS PARA OUTROS ANOS
================================================================================

Para carregar dados para outros anos (2023, 2024, 2025, etc.), use os
scripts .bat criados:

1. Para 2022:
   CARREGAR_DADOS_FINANCEIRO_2022.bat

2. Para 2023:
   CARREGAR_DADOS_FINANCEIRO_2023.bat

3. Para 2024:
   CARREGAR_DADOS_FINANCEIRO_2024.bat

4. Para o ano atual (com parâmetros customizados):
   CARREGAR_DADOS_FINANCEIRO.bat

================================================================================

✅ 67 fornecedores reais (da sua listagem de notas fiscais)

================================================================================
O QUE APARECE NO DASHBOARD
================================================================================

O dashboard mostra dados baseados no PERÍODO FILTRADO:

1. RECEITAS NO PERÍODO: Soma das receitas no período filtrado
2. DESPESAS NO PERÍODO: Soma das despesas no período filtrado
3. SALDO LÍQUIDO: Receitas - Despesas no período
4. FLUXO DE CAIXA DIÁRIO: Gráfico diário do período filtrado
5. EVOLUÇÃO MENSAL: Gráfico dos últimos 12 meses (calculado a partir de hoje)
6. SALDOS POR CONTA: Saldo acumulado de TODAS as contas (não filtra por período)
7. DESPESAS POR FORNECEDOR: Despesas do período filtrado agrupadas por fornecedor
8. DESPESAS POR CENTRO DE CUSTO: Despesas do período filtrado agrupadas por centro

================================================================================
IMPORTANTE
================================================================================

⚠️ Os "Saldos por Conta" mostram o SALDO TOTAL acumulado de todas as contas,
   independente do período filtrado. Por isso você vê valores altos mesmo
   quando o período filtrado mostra R$ 0,00.

⚠️ O gráfico "Evolução Mensal" mostra os ÚLTIMOS 12 MESES a partir de HOJE.
   Se você carregou dados para 2022 e hoje é 2025, o gráfico não mostrará
   os dados de 2022 automaticamente.

⚠️ Para ver os dados de 2022, você DEVE ajustar o filtro de data para 2022.

================================================================================
DICAS
================================================================================

1. Use os botões "30 dias", "90 dias" e "Mês atual" para filtros rápidos
   (mas lembre-se que eles filtram a partir de HOJE, não de 2022)

2. Para ver um mês específico de 2022:
   - Data Início: 01/01/2022
   - Data Fim: 31/01/2022
   - Clique em "Filtrar"

3. Para ver todo o ano de 2022:
   - Data Início: 01/01/2022
   - Data Fim: 31/12/2022
   - Clique em "Filtrar"

4. Para carregar dados para o ano atual (2025):
   - Execute: CARREGAR_DADOS_FINANCEIRO.bat
   - Ou crie um novo .bat para 2025

================================================================================
VERIFICAR SE OS DADOS FORAM CARREGADOS
================================================================================

Para verificar se os dados estão no banco, você pode:

1. Ir em "Financeiro" > "Operações" > "Lançamentos"
2. Filtrar por data: 01/01/2022 a 31/12/2022
3. Você deve ver milhares de lançamentos listados

Ou verificar no dashboard:
1. Ajustar filtro para 2022
2. Verificar se aparecem receitas e despesas
3. Verificar se os gráficos mostram dados
