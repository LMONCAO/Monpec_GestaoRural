============================================================
COMO INICIAR O SISTEMA CORRETAMENTE
MARCELO SANGUINO / FAZENDA CANTA GALO
============================================================

IMPORTANTE: Sempre use o script correto para garantir que o sistema
inicie com o banco de dados correto (Marcelo Sanguino / Fazenda Canta Galo).

============================================================
SCRIPTS DISPONIVEIS
============================================================

1. INICIAR_SISTEMA_MARCELO_SANGUINO_DEFINITIVO.bat
   - RECOMENDADO: Use sempre este script
   - Verifica o banco de dados antes de iniciar
   - Garante que o sistema use o banco correto
   - Para o sistema se o banco estiver incorreto

2. INICIAR_SISTEMA.bat
   - Script principal que redireciona para o script definitivo
   - Use este se preferir um nome mais simples

3. FORCAR_INICIO_MARCELO_SANGUINO.bat
   - Versao alternativa que tambem verifica o banco

============================================================
COMO USAR
============================================================

1. Feche TODAS as janelas do sistema que estiverem abertas

2. Clique duas vezes em:
   INICIAR_SISTEMA_MARCELO_SANGUINO_DEFINITIVO.bat

3. Aguarde a verificacao do banco de dados
   - Se aparecer "[OK] Banco de dados CORRETO confirmado!"
   - O sistema esta pronto para iniciar

4. Se aparecer "[ERRO] BANCO DE DADOS INCORRETO!"
   - NAO continue
   - Verifique se o arquivo db.sqlite3 esta correto
   - Restaure o backup do banco correto se necessario

5. Apos o servidor iniciar, acesse:
   http://localhost:8000/login/

============================================================
VERIFICACAO DO BANCO
============================================================

O sistema verifica automaticamente se o banco contem:
- Produtor: Marcelo Sanguino
- Fazenda: Canta Galo (ou FAZENDA CANTA GALO)

Se nao encontrar, o sistema NAO inicia e mostra um erro.

============================================================
PROBLEMAS COMUNS
============================================================

PROBLEMA: Sistema inicia com banco errado
SOLUCAO: 
  1. Pare TODOS os processos Python
  2. Use o script INICIAR_SISTEMA_MARCELO_SANGUINO_DEFINITIVO.bat
  3. Verifique se o arquivo db.sqlite3 esta correto

PROBLEMA: Aparece landing page ao inves de login
SOLUCAO:
  1. A landing page redireciona automaticamente para /login/
  2. Se nao redirecionar, acesse diretamente: http://localhost:8000/login/
  3. Limpe o cache do navegador (Ctrl+Shift+Delete)

PROBLEMA: Erro ao verificar banco
SOLUCAO:
  1. Verifique se o arquivo db.sqlite3 existe
  2. Execute manualmente: python verificar_banco_correto.py
  3. Se o erro persistir, restaure o backup do banco
