    Sobre o SCR

    O Sistema de Informações de Crédito Rural (SCR) é uma base de dados do Banco Central que registra todas as operações de crédito rural.

    O que será extraído:
    • Dívidas por banco (Sicredi, Banco do Brasil, etc.)
    • Valores a vencer e vencidos
    • Quantidade de contratos
    • Data de referência
    • CNPJ do produtor
    Como obter o SCR:
    1. Acesse o site do Banco Central
    2. Vá em "SCR - Sistema de Informações de Crédito Rural"
    3. Faça login com seu CPF/CNPJ
    4. Baixe o relatório em PDF
    Processamento Automático
    IA Inteligente: O sistema irá automaticamente:
    • Ler o PDF do SCR
    • Extrair dados de dívidas
    • Organizar por banco
    • Separar a vencer/vencido
    • Salvar no sistema
    Importante: Após a importação, você poderá distribuir as dívidas para fazendas específicas.
