Como Verificar o Status de Conformidade

Para verificar se um animal está conforme após a importação:

  • Animais Conformes: Encontrados no BND/SISBOV com dados idênticos (sexo, raça, data de nascimento). Estes animais estão OK e atualizados.
  • Animais Divergentes: Encontrados no BND/SISBOV mas com dados diferentes. Clique em "Ver" para revisar e corrigir manualmente se necessário.
  • Animais Não Conformes: Estão no sistema mas NÃO foram encontrados no BND/SISBOV. Verifique se estes animais realmente existem ou se precisam ser removidos do sistema.
  • Animais Criados: Novos animais adicionados ao sistema a partir do BND/SISBOV.
