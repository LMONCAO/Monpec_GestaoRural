================================================================================
GUIA RÁPIDO DE ACESSO - SISTEMA MARCELO SANGUINO
================================================================================

1. INICIAR O SISTEMA
   ------------------
   Execute: INICIAR_SISTEMA_MARCELO_SANGUINO.bat
   
   Ou: MONPEC DESENVOLVIMENTO.bat

2. ACESSAR NO NAVEGADOR
   ---------------------
   URL: http://localhost:8000

3. LOGIN
   ------
   Username: admin
   Senha: (a senha configurada)

4. PRINCIPAIS MÓDULOS
   -------------------
   
   A) DASHBOARD CONSOLIDADO (Principal)
      URL: http://localhost:8000/relatorios-consolidados/
      
      - Selecionar propriedades
      - Filtrar por ano
      - Ver resumo consolidado
      - Gerar relatórios
   
   B) JUSTIFICATIVA DE ENDIVIDAMENTO
      URL: http://localhost:8000/justificativa-endividamento/
      
      - Documentar situação financeira
      - Importar SCR do Banco Central
      - Calcular capacidade de pagamento
      - Gerar relatório de justificativa
   
   C) RELATÓRIOS ESPECÍFICOS
      - Rebanho: /relatorios-consolidados/rebanho/
      - Bens: /relatorios-consolidados/bens/
      - DRE: /relatorios-consolidados/dre/
      - Fluxo de Caixa: /relatorios-consolidados/fluxo-caixa/
      - Completo: /relatorios-consolidados/completo-emprestimo/

5. FLUXO PARA COMPROVAÇÃO DE EMPRÉSTIMO
   -------------------------------------
   
   Passo 1: Acesse o Dashboard Consolidado
   Passo 2: Selecione todas as propriedades
   Passo 3: Escolha o ano (ex: 2024)
   Passo 4: Gere o Relatório Completo
   Passo 5: Acesse Justificativa de Endividamento
   Passo 6: Importe o SCR (PDF)
   Passo 7: Gere o Relatório de Justificativa

6. INFORMAÇÕES IMPORTANTES NA JUSTIFICATIVA
   -----------------------------------------
   
   - Dívida original: R$ 11.000.000,00
   - Renegociações com aumento de valores
   - Queda de preços (2023 - Jul/2024)
   - Compromisso fixo: Fazenda Canta Galo (R$ 8 milhões/ano)
   - Pagamento como avalista (2024): R$ 2.400.000,00
   - Estratégia: Aguardar estabilização do mercado
